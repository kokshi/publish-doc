    pipenvを利用したPython環境構築

    Created: 2023/01/09

    概要

    次の状態を目指します。

    • vscodeで開発可能
    • フォルダ内に仮想環境を持つ
      • ライブラリ等もプロジェクトフォルダに依存する

    環境

    • Windows11
    • VSCode
    • pipenv

    参考

    • Pipenvを使ったPython開発まとめ
    • 【図解】作業が倍速！pipenvの使い方【Python】
    • [Python] Pipenvで仮想環境をプロジェクトディレクトリ配下に作る方法

    pipenvの導入

    $ pip install pipenv

    仮想環境をプロジェクトディレクトリに作る

    環境変数に下記を追加する。

    変数 値
    PIPENV_VENV_IN_PROJECT true

    プロジェクトの初期化

    次のコマンドを実行するとPC内に入っているPythonのバージョンを使用してpipenvの仮想環境がプロジェクトフォルダに作成される。

    $ pipenv --python 3

    [Project Folder]
┣ .venv/
┃ ┣ Lib/
┃ ┣ Scripts/
┃ ┣ .gitignore
┃ ┣ .project
┃ ┗ pyvenv.cfg
┗ Pipfile

    ライブラリのインストール

    $ pipenv install mkdocs

    ※開発環境のみの場合はこちら

    $ pipenv install --dev mkdocs
